## About Me

I'm a CS student interested in backend systems, infrastructure, APIs, databases, networking, and the broader flow of how software systems interact.

A lot of what pulls me toward technology is curiosity about structure:
- why systems are designed the way they are
- how different layers connect together
- where complexity emerges
- how abstractions simplify or hide things

Recently I've been spending time exploring backend architecture, async workflows, deployment, AI-assisted systems, and the gap between prototypes and real-world software.
